Quill

Quill is a completely free and open source text editor. The goal of Quill was to create an extremely lightweight text editor that utilizes many of the same features as leading applications such as VScode, Sublime, etc. While using many of the same features, we try to keep it as minimal and as lightweight as possible to benefit those without beefy computers.
